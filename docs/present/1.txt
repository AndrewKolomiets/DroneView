Удалось построить тестовую сцену ДТП из 2х автомобилей, дроном сделали видео. Далее посредством программы MetaShape или RealtyCapture(UnrealEngine)
скомбинировали все кадры и на выходе получили 3д сцену. Результат довольно неплохой для первого раза(видео), теперь понятно как правильно облетать сцену, 
не по одной траектории, а каждый виток с поднятием по высоте, должно быть намного лучше. 
Я считаю что наличие 3д сцены ДТП нам действительно поможет сделать интересный продукт, просто извлечение геометрии из фото автоматически пока выглядит нереально (мне пока неизвестно как).
Камера с LIDAR-ом даст намного лучший результат(изучить что есть на рынке из недорогого, нужно попробовать).
Плюс желательно поляризационный фильтр на камеру.
MetaShape разработчики из рф, платная. RealCapture EpicGames(UnrealEngine) бесплатная, насколько известно. Есть OpenSource решения - "CloudCompare". 
Теперь нужно это заиспользовать при анализе сцены.
Нужно только правильно потом разрезать полученую сцену на отдельные обьекты
